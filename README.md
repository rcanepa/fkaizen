# fkaizen

## Installation

* Fork this repository.
* `$ git clone git@github.com:<your username>/fkaizen.git`
* `$ mkvirtualenv fkaizen`
* `$ cd fkaizen/`
* `$ pip install -r requirements.txt`
* `$ npm install`
* `$ bower install`
* `$ python manage.py migrate`
* `$ python manage.py runserver`
